This project is all about creating a map that shows the political leanings of U.S. circuit and district courts. It's designed to help you explore how each court leans, track current judicial vacancies, and see which judges might be retiring soon. My goal is to present all this info in a way that's easy to understand and fun to explore.

Motivation

I'm really into map-making and politics, so this project is a way for me to combine those two interests. The idea came from wanting to make a tool that gives people a better understanding of how the courts are shaping the political landscape in the U.S.

Get Started

On the top right you will see two dropdowns:

The first one will allow you to choose the court type:


                a. District Courts.
                b. Circuit Courts.
The second one will allow you to choose the map type: 

                1. Partisanship: Makeup of courts
                2. Vacancies: Courts with seat(s) that are currently vacant
                3. Retirements: Courts with judges retiring soon
                4. No Vacancies: Makeup of courts if vacancies were filled

Future Plans

Currently I have finished all map options except for the retirement mode. I plan on finishing up that mode at some point next month (Feb 2024). Currently I plan on working on the code quality and adding some tests. I plan on automating the process of updating the maps (the goal is to do it automatically each month) at after all of that. Which might take me some time. Ideally this should all be done by late April 2024 but that is to be seen.
